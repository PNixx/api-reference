REES46 API
------------

* Tracks behavior
* Recognize users
* Recommends products
* Search products
* Send emails
* Send web/mobile push
* etc.

Read the https://reference.api.rees46.com for usage details.

## Setup for updates

Install `rvm`: https://rvm.io

Install ruby 2.3.x using rvm:

```shell
rvm install 2.3.1
```

Use ruby 2.3.1:

```
rvm 2.3.1
```

Install gem bundler, if not present:


```
gem install bundler
```

Install gems:

```
bundle install
```

Run dev server:

```
bundle exec middleman server
```
